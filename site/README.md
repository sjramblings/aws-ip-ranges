# aws-ip-ranges site

Modern editorial dashboard for the daily-mirrored `ip-ranges.json` snapshots in this repo.
Lives at **https://sjramblings.github.io/aws-ip-ranges/** once Pages is enabled.

## Stack

- **Bun** runtime + package manager
- **Vite + React 18 + TypeScript (strict)**
- **Tailwind CSS** with a custom dark editorial palette
- **Recharts** for stacked-area / line charts
- **D3** (selective imports) for the bespoke timeline + calendar heatmap
- **TanStack Table** for the prefix explorer
- **Framer Motion** for entrance animations

## Local development

```bash
# from repo root, generate the time-series JSON from git history
bun tools/extract-history.ts

# then run the site
cd site
bun install
bun run dev      # http://localhost:5173/aws-ip-ranges/
```

## Deploy

`.github/workflows/build-site.yml` runs after every successful `Generate IP Info`
run (or directly on push to `main` touching site/tools), rebuilds the JSON time
series, builds the Vite static bundle, and deploys to GitHub Pages.

To enable Pages on the repo:

1. Settings → Pages → Source: **GitHub Actions**
2. Merge this PR (or push to main); the workflow runs automatically.

## Sections

1. **Hero** — live count-ups, sparkline, last-update pill
2. **Timeline** — D3 brushable area chart with major-event markers
3. **Region Atlas** — 43-region heatmap grid + selectable mini-line chart
4. **Service Composition** — stacked area of top-10 services + long-tail crossings
5. **Compliance & Edge Lens** — Commercial / GovCloud / China / Global YoY growth
6. **Activity Calendar** — 365-day diff-size heatmap (GitHub-style)
7. **Prefix Explorer** — searchable, filterable, sortable table of all prefixes + CSV export
8. **Stories** — auto-generated narrative cards from `events.json`

## Data files

Generated by `tools/extract-history.ts` and served from `public/data/`:

- `timeline.json` — per-commit aggregates (date, total, region_count, diff_size, is_major)
- `regions.json` — per-region time series of prefix counts
- `services.json` — per-service time series of prefix counts
- `events.json` — major commits + region/service first-appearances with narrative
- `current.json` — full latest snapshot for the prefix explorer
